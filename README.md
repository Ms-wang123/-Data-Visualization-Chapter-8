# 第八章 - 数据可视化展示 (增强版)

这是一个功能完善的交互式数据可视化仪表板，完整呈现第8章中的十个可视化图表，并新增了拖放数据输入、样式自定义、图表类型选择等高级功能。

## 🆕 新增功能

### 🎯 拖放式数据输入
- **文件支持**: CSV、JSON、Excel格式
- **智能解析**: 自动识别数据结构并匹配合适图表
- **实时预览**: 拖放后即时显示数据预览
- **灵活处理**: 支持多文件同时导入

### 🎨 样式自定义系统
- **颜色主题**: 5种预设主题 + 自定义颜色选择器
- **字体设置**: 7种中英文字体 + 大小粗细调节
- **布局控制**: 图表间距、圆角、阴影效果自定义
- **预设保存**: 支持保存和加载样式配置

### 📊 图表类型选择器
- **动态切换**: 6种图表类型实时转换
- **智能适配**: 根据数据特征推荐最适合的图表类型
- **保持数据**: 切换类型时保留原始数据

### 🎭 修复与优化
- **等高线图**: 修复坐标轴显示和颜色映射
- **树状图**: 增强聚类算法和数据可视化
- **响应式**: 全面优化移动端体验
- **性能**: 优化大数据量处理和渲染性能

## 📋 功能特性

### 十种完整图表
1. **等高线图** - 可调密度和层级
2. **矢量场流线图** - 流线密度控制
3. **棉棒图** - 数据标签开关
4. **哑铃图** - 年度对比分析
5. **甘特图** - 项目时间管理
6. **人口金字塔** - 年龄结构可视化
7. **漏斗图** - 转化流程分析
8. **桑基图** - 数据流向展示
9. **树状图** - 层次聚类分析
10. **华夫饼图** - 比例数据展示

### 交互控制
- **图表筛选**: 下拉菜单选择显示
- **数据范围**: 滑块实时调整
- **参数控制**: 每个图表独立调节
- **自定义数据**: JSON格式数据导入
- **导出功能**: PNG格式图片下载

### 响应式设计
- **设备适配**: 桌面、平板、手机全覆盖
- **触摸优化**: 移动端手势支持
- **性能优化**: 渐进式加载和渲染
- **无障碍**: ARIA标签和键盘导航

## 🛠 技术栈

### 核心库
- **Chart.js**: 传统图表渲染
- **Plotly**: 复杂可视化支持
- **D3.js**: 数据驱动文档
- **原生JavaScript**: 交互逻辑实现

### 新增技术
- **FileReader API**: 文件拖放处理
- **LocalStorage**: 样式预设存储
- **CSS Grid**: 响应式布局
- **CSS Variables**: 动态主题切换

## 📱 设备支持

### 桌面端 (>1200px)
- 4列网格布局
- 完整控制面板
- 悬停效果和动画

### 平板端 (768px-1200px)
- 2-3列自适应布局
- 紧凑控制面板
- 触摸优化

### 移动端 (<768px)
- 单列垂直布局
- 底部控制面板
- 手势导航

## 🎨 主题系统

### 预设主题
- **Default**: 默认蓝紫渐变
- **Pastel**: 柔和色彩方案
- **Vibrant**: 鲜艳对比配色
- **Monochrome**: 黑白灰单色
- **Rainbow**: 七彩渐变效果

### 自定义选项
- 主题色彩自由选择
- 背景透明度调节
- 文字颜色配置
- 字体家族选择

## 📄 数据格式

### JSON格式
```json
{
  "labels": ["标签1", "标签2"],
  "values": [数值1, 数值2]
}
```

### CSV格式
```csv
类别,数值1,数值2
项目A,100,200
项目B,150,180
```

### 拖放使用
1. 点击"拖放模式"按钮
2. 将文件拖放到指定区域
3. 查看数据预览确认
4. 自动应用到对应图表

## ⚙️ 高级设置

### 图表参数
- **网格密度**: 控制数据精度
- **等高线数量**: 调整显示层级
- **流线密度**: 矢量场细节控制
- **年龄分组**: 人口分析精度

### 性能优化
- **数据采样**: 大数据自动降采样
- **懒加载**: 图表按需渲染
- **缓存机制**: 样式和数据缓存
- **内存管理**: 及时释放资源

## 🔧 使用方法

### 基础使用
1. 用浏览器打开 `第八章.html`
2. 选择要显示的图表类型
3. 调整数据范围和参数
4. 查看交互式图表

### 高级功能
1. 点击"拖放模式"导入外部数据
2. 使用"样式自定义"个性化界面
3. 通过图表类型选择器切换显示方式
4. 导出高质量图片文件

### 样式预设
1. 调整各项样式参数
2. 点击"保存预设"存储配置
3. 下次访问自动加载设置
